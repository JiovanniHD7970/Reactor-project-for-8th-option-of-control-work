# Reactor-project-for-8th-option-of-control-work
Санкт-Петербургский Государственный Технологический Институт

Разработка программных комплексов для исследований в химии и химической технологии 

Вариант 8 

# Запуск приложения

1. Скачайте или клонируйте репозиторий.
2. Перейдите в корневую папку проекта.
3. Запустите файл `realese/app.exe` (двойной клик мышью).
Важно: приложение не требует установленного Python, так как все необходимые компоненты уже упакованы в realese/app.exe.

# Учётные записи

При первом запуске в базе данных уже созданы два пользователя:

- **Администратор**
  - Логин: `admin`
  - Пароль: `admin`

- **Исследователь**
  - Логин: `user`
  - Пароль: `1234`

Роль администратора предназначена для настройки библиотеки коэффициентов:
- добавление новых типов сырья;
- редактирование параметров модели и диапазонов варьирования.
- полный функционал исследователя

Роль исследователя предназначена для проведения расчётов:
- выбор типа сырья;
- загрузка параметров модели из БД;
- запуск расчёта;
- просмотр таблицы и графиков;
- сохранение отчёта в формате CSV.
